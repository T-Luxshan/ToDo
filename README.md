# React To-Do App

A simple To-Do application built using React.

## Features
- Add tasks
- Mark tasks as completed.
- Delete tasks.
- Fetch tasks from a backend API.

   ```bash
   git clone https://github.com/T-Luxshan/ToDo.git
   cd <repository-directory>
